# Data Directory

This directory contains all data files for the Ridewise Europe project.

## Structure

- **raw/**: Original, unprocessed data files (CSV format)

  - `customers.csv`: Customer profiles and demographics
  - `trips.csv`: Trip records with fare and revenue data
  - `promotions.csv`: Available promotions
  - `promo_redemptions.csv`: Promotion usage tracking
  - `customer_activity.csv`: Customer app activity and funnel events

- **processed/**: Cleaned and feature-engineered datasets ready for modeling
  - Generated by `src/data/build_features.py`

## Data Generation

To generate raw data files, run:

```bash
python src/data/generate_data.py
```

This will create all CSV files in the `raw/` directory.
